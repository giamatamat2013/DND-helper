// CHAPTER I

חזון המערכת

יצירת "מרכז פיקוד" דיגיטלי לקבוצת חברים המשחקים משחק תפקידים סיפורי. המערכת הופכת את השיח החי — קול, וידאו, צ'אט — לנכס דיגיטלי מתועד, מאורגן ומנוהל על ידי בינה מלאכותית, תוך שילוב לוח טקטי אינטראקטיבי. כל שיחה, כל קרב, כל החלטה עלילתית — נשמרת, מסוכמת ומוכנה להמשך.

Phase: Design
APIs: Pending Config
Backend: Not Deployed
Target: GitHub Pages
⚔
// CHAPTER II

ארבעת עמודי התווך

כל מערכת נשענת על ארבעה ממדים — תקשורת, אינטליגנציה, ארכיון, לוח טקטי.

📡
// PILLAR I
ערוצי תקשורת
  • צ'אט רב-ערוצי לפי נושאים (עלילה, חוקים, לור)
  • הודעות קוליות, סרטונים, תמונות וקבצים
  • שיחת וידאו / אודיו לייב עם הקלטה
  • הקלטה נפרדת לכל משתתף (לתמלול)
  • מטא-דאטה אוטומטי: שולח, זמן, ערוץ
🧠
// PILLAR II
המוח — AI Intelligence
  • הכרוניקן: תמלולים → סיפורים מובנים
  • ייצוא PDF/Markdown/Word לדרייב
  • ממשק פקודות טבעי לבקשות מה-AI
  • זיהוי הקשר לפי ערוץ (מפות / סיפור)
  • ניתוח תמונות, ניתוח עלילה, סיכומי קרב
🗄️
// PILLAR III
המאגר — The Archive
  • Firebase Firestore לנתונים בזמן אמת
  • חיבור ישיר ל-Google Drive
  • כספת קבצים — מסונן לפי תאריך ונושא
  • שמירת מצב מלאה (State Persistence)
  • המשכיות — חזרה לנקודה כלשהי בעבר
⚔️
// PILLAR IV
הלוח הטקטי
  • מפה אינטראקטיבית בזמן אמת (Multiplayer)
  • כלי ציור: חצים, אזורים, צבעים
  • שכבות: רקע, סימוני שחקנים, DM
  • סנכרון עם Firebase RTDB
  • ייצוא תמונת מצב למסמך הסיכום

// CHAPTER IV — הגדרות

מפת ה-APIs וההגדרות

כל שירות חיצוני שהמערכת זקוקה לו, עלותו, ואופן ההגדרה.

🔥 Firebase (Google) — ליבת ה-Backend
שירות מטרה עלות סטטוס
Firebase Auth התחברות עם Google / Email לכל שחקן Free Tier (10K/month) חינם
Firestore נתוני צ'אט, פרופילי שחקנים, מטא-דאטה 50K reads / 20K writes free/day חינם
Realtime Database מצב הלוח הטקטי בזמן אמת 1GB storage, 10GB transfer free חינם
Firebase Storage קבצים, הקלטות, תמונות מפות 5GB free חינם
Firebase Functions לוגיקת שרת (AI Calls, Drive export) 125K invocations free / לא זמין בGitHub Pages שרת נפרד
→ console.firebase.google.com | צור Project חדש | הפעל Authentication + Firestore + RTDB
🧠 AI / LLM — הכרוניקן
שירות מטרה עלות סטטוס
Anthropic Claude API סיכומים, ניתוח עלילה, יצירת מסמכים ~$3 / 1M tokens (Sonnet) API Key
OpenAI GPT-4o אלטרנטיבה, ניתוח תמונות מפות ~$5 / 1M tokens אופציונלי
OpenAI Whisper Speech-to-Text להקלטות $0.006 / minute API Key
Web Speech API Speech-to-Text בדפדפן (ללא עלות) Chrome/Edge בלבד חינם
⚠ API Keys חייבים לגור ב-Firebase Functions בלבד — לעולם לא בקוד Frontend!
📹 וידאו ואודיו לייב
שירות מטרה עלות סטטוס
Daily.co SDK וידאו + אודיו P2P, הקלטה לכל משתתף Free: 10K min/month | $0.004/min beyond מומלץ
LiveKit WebRTC עצמי, open-source, self-hosted Self-hosted = חינם, Cloud = $ אלטרנטיבה
WebRTC Native P2P ישיר בין שחקנים (ללא שרת) STUN/TURN חינם + TURN שרת $ מורכב
📂 Google Drive — The Archive
שירות מטרה עלות סטטוס
Google Drive API v3 העלאה/הורדה אוטומטית של קבצי AI Free (בתוך quota) OAuth2
Google Docs API יצירת מסמכי סיכום ישירות Free אופציונלי
→ console.cloud.google.com | Enable Drive API | OAuth 2.0 Credentials | Scopes: drive.file
